Cómo calcular el tiempo que tarda un script PHP en ejecutarse

Hace tiempo tenía rondando una idea por mi cabeza, la de calcular cuánto tiempo tarda un script PHP en ejecutarse. Bueno vale, no era una idea mía, es algo que ves en casi todos los foros como PHPbb, IPB, vB, étc.

Esto me sirvió para depurar algunos scripts y hacerlos mas rápidos, ya que aunque es variable, siempre suele estar en unos márgenes muy precisos. Para hacerlo, nos basta con usar microtime.

$time = explode(' ', microtime());
$start = $time[1] + $time[0];

// Aquí pon tu código.

$time = explode(' ', microtime());
$finish = $time[1] + $time[0];
$totaltime = ($finish - $start);
echo 'Tiempo total en ejecutar el script: '.$totaltime;

Para probarlo, usad un bucle por ejemplo, o algo que tarde en ejecutarse, no vale con un simple “echo”.
